Scope and Objectives

Build and test a hybrid deep learning model to improve stock price prediction
Combine three data types: financial data, news sentiment, and past prices
Data collection, cleaning, and preprocessing
Design an LSTM-based model with FinBERT for sentiment analysis
Compare results with standard prediction models

Background and Previous Work

Most existing models use only one data type (technical, fundamental, or sentiment)
Recent research suggests combining all three improves accuracy
Innovation: LSTM network for time-series + FinBERT for financial text analysis
Expected to outperform traditional methods in prediction accuracy

Proposed Methodology

1. Collect and clean data: prices, financials, news
2. Use FinBERT to analyze news and create sentiment scores
3. Build multi-input LSTM model combining all data types
4. Test accuracy using MAE and RMSE, compare with baseline

Hybrid Model Architecture

Input Layer: Historical Prices (Time Series Data)
Input Layer: Company Fundamentals (Structured Data)
Input Layer: News Sentiment Scores (from FinBERT)
LSTM Layers for Temporal Patterns
Dense Layers for Feature Combination
Output Layer: Price Prediction

Expected Benefits

A practical stock prediction tool for investors
Demonstration that multi-data approaches improve accuracy
Contribution to financial data science methodology
Potential for real-world application in trading systems
Foundation for future research in hybrid models
